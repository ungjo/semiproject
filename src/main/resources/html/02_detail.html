Spring Boot와 MyBatis 연동하는 방법 정리

작성자: hong 날짜: 2025-05-31 10:23:45 조회수: 43
Spring Boot와 MyBatis를 연동하려면 먼저 build.gradle에 의존성을 추가해야 합니다. 1. implementation 'org.mybatis.spring.boot:mybatis-spring-boot-starter:3.0.3' 2. runtimeOnly 'com.mysql:mysql-connector-j' 그리고 application.properties에 DB 연결 정보와 MyBatis 설정을 추가합니다. spring.datasource.url=jdbc:mysql://localhost:3306/semiproject mybatis.mapper-locations=classpath:mappers/*.xml mybatis.configuration.map-underscore-to-camel-case=true 마지막으로 Mapper 인터페이스에 @Mapper 어노테이션을 붙이면 완성입니다!
